# Privacy policy GitHub Pages (≈10 મિનિટ)

1. github.com → login → **New repository**  
   - Name: `vidyarthi-lalkitab-privacy`  
   - Public → Create  

2. **Add file** → **Upload files**  
   - Upload: `publish/index.html`  
   - (પહેલાં `.\scripts\sync-publish-config.ps1` ચલાવ્યું હોય તો email સાચો હશે)  
   - Commit  

3. **Settings** → **Pages**  
   - Source: Deploy from branch **main** / **root**  
   - Save  

4. 2–5 મિનિટ પછી URL:  
   `https://YOUR_USERNAME.github.io/vidyarthi-lalkitab-privacy/`  

5. Browser માં ખોલી ચેક કરો.  

6. `publisher.config` માં `privacy.policy.url` = આ URL  
7. ફરી: `.\scripts\sync-publish-config.ps1`
