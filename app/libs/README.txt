guava-31.1-android.jar
========================
Bundled locally because some PCs get "Tag mismatch" when Gradle downloads from Maven
(often antivirus HTTPS scanning or unstable internet).

If you delete this file, restore it:
https://repo.maven.apache.org/maven2/com/google/guava/guava/31.1-android/guava-31.1-android.jar

Or run:  .\scripts\download-guava.ps1
